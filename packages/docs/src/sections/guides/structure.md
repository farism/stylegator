# Directory Structure

The recommended directory structure for the default configuration:

```sh
my-app/
│
├── components
│   └── MyComponent.js
│
├── partials
│   ├── Credits.js
│   └── Logo.js
│
├── public
|   └── assets/
│       └── logo.png
│
├── sections/
│   ├── introduction.md
│   └── getting-started.md
│
├── index.js
└── index.html
```

You can configure this project structure via the CLI or the `stylegator.config.js` file.
